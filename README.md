# 🌍 Decentralized Collaborative Itinerary Planning

Welcome to a revolutionary way to plan group travel itineraries on the Stacks blockchain! This project enables groups to collaboratively create, vote on, and manage travel itineraries with transparency and trust, using Clarity smart contracts.

## ✨ Features

- **Collaborative Planning**: Propose and vote on itinerary activities (e.g., flights, accommodations, events).
- **Immutable Records**: Store itinerary details and voting history on the blockchain.
- **Secure Payments**: Manage shared expenses with escrow-like functionality.
- **Contribution Tracking**: Ensure fair cost-sharing among group members.
- **Access Control**: Restrict itinerary modifications to authorized group members.
- **Dispute Resolution**: Handle disagreements through a transparent voting mechanism.
- **Verification**: Allow external parties (e.g., travel agencies) to verify finalized itineraries.

## 🛠 How It Works

### For Group Members
1. **Create a Group**: Start a new travel group and invite members using their Stacks addresses.
2. **Propose Activities**: Submit itinerary items (e.g., hotel bookings, activities) with details like cost and description.
3. **Vote on Proposals**: Group members vote to approve or reject proposed activities.
4. **Contribute Funds**: Deposit STX for shared expenses, held in escrow until the itinerary is finalized.
5. **Finalize Itinerary**: Once approved, lock the itinerary and release funds to service providers.
6. **Resolve Disputes**: Use a voting mechanism to settle disagreements on itinerary changes.

### For Verifiers
- Query itinerary details to confirm finalized plans or group contributions.
- Verify payment status for booked services.
